# 🎓 AI Student Support Chatbot

An AI-powered Student Support Chatbot built using **Python (Flask), HTML, CSS, and JavaScript**. It helps students get instant answers related to admissions, courses, exams, and college information through an interactive chat interface.

---

## 🚀 Features

- 💬 AI-powered chatbot
- 🌙 Dark Mode / Light Mode
- 🗑️ Clear Chat option
- 🎤 Voice Input Support
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly design

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Git & GitHub

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yogesh-shukla-5635/AI-Student-Chatbot.git
```

2. Go to the project folder

```bash
cd AI-Student-Chatbot
```

3. Install dependencies

```bash
pip install flask
```

4. Run the project

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
